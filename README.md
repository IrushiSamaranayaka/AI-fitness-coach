 🏋️ AI Fitness Coach Web App

A modern AI Fitness Coach web application that generates personalized workout plans using user data such as BMI, fitness goals, and experience level.


🚀 Features
 🧑‍💻 User input form (Name, Goal, Experience, Workout Days)
 📊 BMI calculation using height and weight
 🏋️ Personalized fitness plan generation
 🥗 Nutrition tips and motivation messages
 📥 Download plan feature 
 🎨 Clean and modern UI


🛠️ Technologies Used
 HTML
 CSS
 JavaScript

 🎯 How to Run
1. Download or clone the repository  
2. Open `index.html` in your browser  
3. Enter your details and generate your fitness plan
